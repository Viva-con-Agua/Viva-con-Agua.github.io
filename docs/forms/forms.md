# Form Components

#### Root Element for a Form

```
<vca-form>
content
</vca-form>
```

#### Define a field Element

```
<vca-field label="Label">
    content
</vca-field>
```

#### Content will display in a row
```
<vca-field-row>
    content
</vca-field-row>
```
