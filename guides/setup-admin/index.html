Setup administrator

Setting up an user as administrator requires access to Drops database. Connect to the database docker using 

docker exec -it drops-mariadb mysql -u drops -p

enter the password of the drops database user and switch to the database used by Drops (default: drops). Select all user saved in the database 

SELECT * FROM User;

Now, choose the user that has to be administrator and update the user with the following command:

UPDATE User SET `roles` = 'supporter,admin' WHERE `id` = 1;

(replace the id with the ID of teh chosen user!)
